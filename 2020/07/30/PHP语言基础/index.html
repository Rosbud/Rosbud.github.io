这篇文章简单对PHP的各种常用基础知识进行介绍，以理解为重点，省略大部分运行情况

语言基础

  数据结构

    数据类型

  1.1 boolean

    $boo = true 将变量命名为布尔型
  1.2 string

    $a = '字符串' 单引号的变量按普通字符串输出

    $a = "字符串" 双引号中包含的变量会被自动替换成实际数值

    $string = <<<str 要输出的字符 str 输出字符，效果同双引号

  1.3 integer

    可以声明十进制、八进制和十六进制整数

  1.4 float

    提供两种书写格式：标准格式和科学计数法（例 3.58E1,87E-3）

  1.5 array

    一组数据的集合

    语法格式：$array = array('value1','value2') 或 $array[key]='value' 或 $array = array(key1 => value1,key2=>value2)

    其中 key 是数组元素的下标， value 是数组下标对应的元素

  1.6 object

    面向对象和面向过程

  1.7 特殊数据类型

  资源（resource）和空值（null）

>资源：又叫句柄，保存了到外部资源的一个引用
>空值：表示变量没有值
>unset()：销毁指定的变量

 

    数据类型转换

    变量前加上 小括号 括起来的类型即可

(boolean)$sum 、(inter)$boo

    函数 settype（） 完成

bool settype(mixed var,string type)

    var —— 指定的变量

    type —— 指定的类型（boolean、float、integer、array、null、object、string）

    转换成功返回 true 失败则 false

 

    检测数据类型

    PHP内有许多内置函数关于检测，为 is 系列

is_bool ; is_string ; is_float ; is_int ; is_null ; is_array ; 

is_null ; is_object ; is_numeric(检测是否为数字或者由数字组成的字符串)

 

  常量

    声明和使用

  使用 define() 来定常量

define(string constant_name,mixed value,case_sensitive=false)

constant_name : 必选参数，常量名称，即标识符

value : 必选参数，常量的值

case_sensitive : 可选参数，指定是否大小写敏感，true则不敏感

  获取常量设置的值有两种方法，一种是使用常量名输出，另一种是使用函数 constant() 输出

mixed constant(string const_name)

const_name : 常量名，成功则返回常量的值，失败则输出错误

  判断常量是否被定义可以使用 defined() 函数

bool defined(string constant_name)

constant_name : 常量名称，存在则返回true，否则是false

 

    预定义常量

  PHP中有系统预定义常量

__FILE__ : php程序文件名
__LINE__ : php程序行数
PHP_VERSION : php版本
PHP_OS : 执行解析php的操作系统
TRUE or FALSE or NULL : 布尔值和空值
E_ERROR : 指到最近错误处
E_WARNING : 指到最近警告处
E_PARSE : 指到解析语法有潜在问题处
E_NOTICE : 不寻常处提示

 

  PHP变量

 

    变量声明及使用

  变量名称由 $ 和 标识符 组成。标识符由字母、数字或下划线组成，且不能以数字开头。

  变量赋值一般有直接赋值，还有变量间赋值。

直接赋值和其他语言一样
引用赋值在变量前加上 & 符号取其他变量的地址

区别：直接赋值是开辟新的内存空间，引用赋值则是多一个名字

 

    变量作用域

  php和其他语言一样有局部变量，全局变量(global)和静态变量(static)

全局变量：被定义在所有函数之外
静态变量：能够在函数调用结束后仍然保留变量值，再次回到作用域时，可继续使用

 

    可变变量

  允许动态改变变量名称（套娃变量）

$animal = 'turtles';      $turtles = 103;
echo $$animal;
则会输出103数字

 

    PHP预定义变量

$_GET — HTTP GET 变量
$_POST — HTTP POST 变量
$_FILES — HTTP 文件上传变量
$_REQUEST — HTTP Request 变量
$_SESSION — Session 变量
$_COOKIE — HTTP Cookies，多数由setcookie()函数设置

$_SERVER['REMOTE_ADDR'] —— 浏览当前页面的用户的 IP 地址
$_SERVER['SERVER_ADDR'] —— 当前运行脚本所在的服务器的 IP 地址。
$_SERVER['SERVER_NAME'] —— 当前运行脚本所在的服务器的主机名
$_SERVER['SERVER_PORT'] —— 服务器使用的端口
$_SERVER['DOCUMENT_ROOT'] —— 当前运行脚本所在的文档根目录。在服务器配置文件中定义。
$_SERVER['PHP_SELF'] —— 获取当前网页地址（不含域名部分）
$_SERVER['QUERY_STRING'] —— 获取当前网页地址中的所有get数据（就是？号后面的部分），但只是一个整体的字符串而已
$_SERVER['SCRIPT_FILENAME'] —— 当前执行脚本的绝对路径
$_SERVER['SERVER_SIGNATURE'] —— 包含服务器版本和虚拟主机名的字符串

  PHP函数

    定义和调用函数

  函数，即将重复调用的功能卸载独立一个模块内

创建函数基本语法格式：
function fun_name($str1,...$strn){
    fun_body
}
function    为声明自定义函数时必须使用的关键字
fun_name    为自定义函数名
$strn        为函数参数

 

    函数间传递参数

  1. 按值传递： 传参在函数定义域内值改变不影响外部

  2. 按引用传递： 传参时加上 & 将参数内存地址传入，函数定义域内外参数的值都会改变

  3. 默认参数 ： 一般放在参数列表末尾，并指定默认值
 

    函数返回值

  用 return 返回一个值，只能一个值，若要多个则需要创建数组

    变量函数

  基本效果如同可变变量一样（套娃）

  字符串操作

    去除字符串特殊字符和首尾空格

  对用户输入字符进行过滤空格和特殊字符，有 trim() , ltrim() ，rtrim() 来去除

  1.trim() 函数

string trim(string str [,string charlist]);
trim()主要用于过滤字符串首尾字符
str 是要操作的字符串对象；charlist为可选参数，一般列出所有希望过滤的字符

 
  2.ltrim() 函数

string ltrim(string str [,string charlist]);
ltrim()主要用于过滤字符串左边字符，参数和trim()一样

 
  3.rtrim() 函数

string ltrim(string str [,string charlist]);
rtrim()主要用于过滤字符串右边字符，参数和trim()一样

 

    转义和还原字符串数据

  一共有两种转义、还原字符串方式：手工和自动

  1.手动

  使用 \ 将后面第一个字符变得没有特殊意义

  2.自动

  ①addslashes()函数

string addslashes(string str)
addlashes函数是使用反斜线引用字符串
str为要转义的字符串，返回值为转义后的字符

 
  ②stripslashes()函数

string stripslashes(string str)
stripslashes函数是反引用一个引用字符串，即将addslashes函数转义后的字符串还原
str为要还原的字符串

 
  ③addcslashes()函数

string addcslashes(string str,string charlist)
addcslashes函数对指定的字符串前加上反斜线
str为将要被操作的字符串，charlist则指定字符串中哪些字符串加反斜线

 
  ④stripcslashes()函数

string stripcslashes(string str)
stripcslashes函数对addcslashes函数转义的字符进行还原
str为要还原的字符串

 

    获取字符串长度

  主要使用 strlen() 函数进行获取

int strlen(string str)
获取字符串长度

 

    截取字符串

  对字符串截取可以采用预定义函数 substr() 函数实现

string substr(string str,int start[,int length])
str     指定字符串对象
start    指定开始截取字符串的位置，若为负数则从字符串末尾开始
length    可选参数，指定截取字符串个数，若为负数则表示取倒数第length个字符

 

    比较字符串

  比较方法主要有三种：按照字节，按照自然排序和指定源字符串位置开始

  ①按照字节进行字符串比较：strcmp() 和 strcasecmp()

int strcmp(string str1 , string str2)
int strcasecmp(string str1 , string str2)
str1和str2为比较的字符串，相等则返回0，str1大于str2返回值大于0，小于则返回小于0
strcmp()和strcasecmp()差别在于：strcmp()区分大小写而strcasecmp()不区分

 
  ②按自然排序法进行比较：strnatcmp()

int strnatcmp(string str1 , string str2)
strnatcmp()将字符串中的字符从左到右顺序进行比较
数字和数字比较按照自然排序法，其他则按照ACSII码值比较
str1和str2为比较的字符，相等返回0，str1比str2大则返回大于0，小则返回小于0

 
  ③指定从源字符串位置开始比较：strncmp()

int strncmp(string str1 , string str2 , int len)
len为必要参数，指定每个字符串中参与比较的字符串数量
strncmp 区分函数大小写
返回值和上面的相同

 

    检索字符串

  ① strstr() 函数

  用于获取一个指定字符串在另一个字符串中首次出现的位置到后者末尾的子字符串，成功则返回剩余字符串，失败则返回false。简单的说就是返回一个字符串某个标识后面所有字符。

string strstr(string haystack ， string needle)
haystack    必要参数，指定从哪个字符串中进行检索
needle        必要参数，指定搜索对象。
strstr()区分字母大小写//大小写绕过

 
  ② substr_count() 函数

  获取指定字符在字符串中出现的次数

int substr_count(string haystack ， string needle)
haystack    指定字符串
needle        指定字符

 

    替换字符串

  ① str_ireplace() 函数

  使用新的字符串替换原始字符串中指定的字符

mixed str_ireplace(mixed search ，mixed replace，mixed subject [，int &count])
search    必要参数，要搜索的值，可以用array来提供多个值
replace    必要参数，指定替换的值
subject    必要参数，要被搜索和替换的字符串或数组
count    可选参数，执行替换的数量
str_ireplace 函数不区分大小写

 
  ② substr_replace() 函数

  用于对指定字符串中的部分字符串进行替换

mixed substr_replace(mixed string，mixed repl，mixed start ，[mixed length])
string    指定要操作的原始字符串
repl    指定要替换后的新字符串
start    指定替换字符串开始的位置
length    可选参数，指定返回的字符串长度，默认是返回整个字符串
若start为负数，而length小于或等于start的值，length的值为0

 

     格式化字符串

  number_format() 函数可用来将数字字符串格式化

string number_format(float number[,int decimals[,string dec_point[,string thousands_sep]]])
number            为要格式化的数字
decimals        为要保留的小数位数
dec_point        为指定小数点显示的字符
thousands_sep    为指定千位分隔符显示的字符

 

    分割和合成字符串

  ① explode() 函数按照一定规则对一个字符串进行分割

array explode(string delimiter,string str[,int limit])
delimiter    边界上的分隔符
str            必要参数，指定将要被分割的字符串
limit        可选参数，若设置了limit参数，则返回的数组包含最多limit个字符
注意：返回是以数组形式

 
  ② implode() 函数将数组内的内容合成一个新的字符串

string implode(string glue,array pieces)
glue    字符串类型
pieces    数组类型，指定要被合并的数组

 

正则表达式

  正则表达式的概念

  正则表达式是一种描述字符串结构的语法规则，是一种特定的格式化模式，可以匹配、替换、截取匹配的字符串，类似dos命令中的 dir *.txt

  这里说明几种术语

grep    最初是ED编辑器的一条命令，用于显示文件中的特定内容，后独立成一个工具
egrep    grep的扩展版，大大增强grep的能力
POSIX    可移植操作系统接口，POSIX是一系列标准，确保操作系统之间的可移植性
Perl    实际抽取与汇报语言，成为与POSIX之后的另一个标准
PCRE    兼容Perl正则表达式的一套正则引擎，PHP正是使用该引擎之一

 

  正则表达式的语法规则

  一个完整的正则表达式由两部分构成，元字符和文本字符。元字符是具有特殊含义的字符，如前面列举dos命令中的 * 符号。文本字符就是普通文本，如数字和字母。PCRE风格的正则表达式一般都是放置在定界符 / 中间。

    行定位符(^ 和 $)

  行定位符就是用来描述字符的边界。__ ^ 表示行的开始， $ 表示行的结尾。__

^tm 该表达式表示要匹配字串 tm 的开始位置是行头
例如： tm this is a dog 就可以匹配，this is a dog tm 则不匹配
tm$ 则可以匹配例子:this is a dog tm
tm 则匹配的字串可以出现在字符串任意位置

 

    单词分界符(\b 和 \B)

  以上面的例子，若按照 tm 则会匹配出 html 内的一部分 tm 而非一个单词 tm ，这是则可使用单词分界符 \b 表示，这表示要查找的是为一个完整的单词。 

\btm\b

  \B 正好相反，表示匹配的是单词或字符串的一部分。

\Btm\B

 

    字符类([ ])

  正则表达式是区分大小写的，若要忽略大小写则使用 [ ]
